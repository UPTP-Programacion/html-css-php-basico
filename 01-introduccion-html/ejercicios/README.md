# Ejercicios · 01 Introducción a HTML

## Ejercicio 1: Mi Tarjeta de Presentación

Crea un archivo `tarjeta.html` que contenga:

- La estructura básica de HTML5 (`<!DOCTYPE html>`, `<html>`, `<head>`, `<body>`)
- Un título en la pestaña: "Mi Tarjeta"
- Un encabezado `<h1>` con tu nombre completo
- Un encabezado `<h2>` que diga "Sobre mí"
- Dos párrafos `<p>` describiendo quién eres y qué estudias
- Una línea horizontal `<hr>` de separación
- Un encabezado `<h3>` que diga "Contacto"
- Un párrafo con tu correo electrónico

### Resultado esperado

La página debe mostrarse correctamente en el navegador con toda la información organizada por secciones.

---

## Ejercicio 2: Página de Mis Materias

Crea un archivo `materias.html` que contenga:

- Estructura HTML5 completa
- Un `<h1>` con "Mis Materias del Semestre"
- Por cada materia que curses, usa un `<h2>` con el nombre y un `<p>` con una breve descripción
- Agrega al menos **4 materias**
- Usa comentarios HTML (`<!-- -->`) para separar cada sección

### Pista
```html
<!-- Materia 1 -->
<h2>Programación I</h2>
<p>En esta materia aprendemos los fundamentos de la programación...</p>
```

---

## ✅ Criterios de evaluación

- [ ] El archivo tiene la estructura HTML5 correcta
- [ ] Se usa `<meta charset="UTF-8">` para caracteres especiales
- [ ] Los encabezados están organizados jerárquicamente (h1 > h2 > h3)
- [ ] El archivo se visualiza correctamente en el navegador
