# Ejercicios · 05 Introducción a CSS

## Ejercicio 1: Estiliza tu Tarjeta de Presentación

Toma el archivo `tarjeta.html` del tema 01 (o crea uno nuevo) y agrega estilos con un archivo CSS externo:

1. Crea un archivo `tarjeta.css` y enlázalo al HTML
2. Aplica los siguientes estilos:
   - Fuente personalizada para todo el body
   - Color de fondo para la página
   - Encabezado `h1` centrado con un color llamativo
   - Párrafos con un tamaño legible y color gris oscuro
   - Un borde redondeado alrededor de la sección de contacto
3. Experimenta con al menos **3 colores diferentes** (hexadecimal, rgb, nombre)

---

## Ejercicio 2: Tres Formas de CSS

Crea un archivo `tres-formas.html` que demuestre las 3 formas de aplicar CSS:

1. Un párrafo con **CSS en línea** (style="...")
2. Un párrafo estilizado con **CSS interno** (`<style>` en el head)
3. Un párrafo estilizado con **CSS externo** (archivo .css)
4. Cada párrafo debe verse diferente
5. Agrega un comentario HTML explicando cada método

---

## Ejercicio 3: Paleta de Colores

Crea una página `paleta.html` con su archivo CSS que muestre:

1. **6 cajas** (divs) cada una con un color de fondo diferente
2. Dentro de cada caja, muestra el código del color (hex, rgb o nombre)
3. El texto debe ser legible sobre cada fondo (contraste)
4. Las cajas deben tener bordes redondeados y un poco de padding
5. Usa al menos 2 formatos de color diferentes (hex y rgb)

### Resultado esperado
Algo similar a una paleta de colores tipo diseño gráfico.

---

## ✅ Criterios de evaluación

- [ ] El CSS externo está correctamente enlazado
- [ ] Se usan propiedades de texto (font-size, color, text-align)
- [ ] Se usan propiedades de fondo (background-color)
- [ ] Se usan bordes y border-radius
- [ ] El código CSS está organizado y comentado
