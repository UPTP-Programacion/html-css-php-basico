## 📖 Descripción

Repositorio educativo que cubre los fundamentos de **HTML**, **CSS** y **PHP** de forma progresiva. Cada tema incluye teoría explicada, ejemplos funcionales y ejercicios prácticos para reforzar el aprendizaje.

> 💡 **Ideal para:** Estudiantes que inician en el desarrollo web desde cero.

```
📦 html-css-php-basico/

├── 01-introduccion-html/
```

## 🚀 ¿Cómo Usar Este Repositorio?

```bash
# 1. Clona el repositorio
git clone https://github.com/UPTP-Programacion/html-css-php-basico.git

# 2. Entra a la carpeta
cd html-css-php-basico

# 3. Navega al tema que estés estudiando
cd 01-introduccion-html

# 4. Abre los archivos en tu editor
code .
```

> 📌 Para los temas de **PHP** (09 en adelante), copia la carpeta dentro de `htdocs/` de XAMPP y accede desde `http://localhost/`.
